# 🏥 Hospital Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Hospital Analytics Dashboard** developed using **Power BI**.
The dashboard provides insights into patient admissions, department workload, revenue trends, and hospital performance metrics.

The goal of this project is to demonstrate **data analysis, visualization, and business intelligence skills** by transforming raw hospital data into meaningful insights.

---

## 🎯 Objectives

* Analyze **patient admission trends**
* Identify **department-wise patient distribution**
* Track **hospital revenue performance**
* Understand **gender and demographic distribution**
* Evaluate **doctor and department performance**

---

## 📊 Dashboard Features

The Power BI dashboard includes the following key components:

### 🔹 KPI Metrics

* Total Patients
* Total Revenue
* Average Patient Age
* Average Length of Stay

### 🔹 Visualizations

* Patient Distribution by Department
* Gender Distribution of Patients
* Monthly Admission Trends
* Revenue by Department
* Top Doctors by Patient Count

### 🔹 Interactive Filters (Slicers)

* Department
* Doctor
* Gender
* Date

These filters allow users to explore the data interactively and gain deeper insights.
